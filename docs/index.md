# wltk (Wayland Toolkits)

## 理念

wltk将是一个完全基于wayland的、现代的图形界面工具库。wltk将使用EGL渲染。

## 组成

### frame[窗口框架]

* [window](frame/window.md)
* [popup](frame/popup.md)

### surface[绘图表面]

### component[组件库]

* [toolsbar]工具栏
* [text]文本
* [inputbox]输入框
* [editor]编辑器
* [botton]按钮
* [choicebox]多选框
* [choicecard]选项卡

### event[事件系统]

### graphics[图形库]

* [base]基本图形
* [transformation]图形变换
* [characters]文本渲染
* [image]图像渲染

### web[网络渲染]
